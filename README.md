# BTA Not Public

Private source repository for The Words We Carry / BTA magazine system.

## Contents

- `figma-source/` — clean GitHub-only reader source for `BTA_Magazine_Not_Tied_to_Figma`.
- `wordpress-plugins/` — WordPress plugin source folders.
- `wordpress-plugin-install-zips/` — installable WordPress plugin zips.
- `docs/` — private architecture/build notes.

## Content rule

Magazine content is not owned here. The public content source is:

`https://raw.githubusercontent.com/Joliel21/bta_public/main/public/content/issue.json`

Edit public content in `Joliel21/bta_public`, then refresh Figma/WordPress. Rebuild this private repo only when reader or plugin code changes.
